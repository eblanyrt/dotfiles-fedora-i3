# eblanyrT's Fedora i3 dotfiles

## Installation
1. Install git:
    ```
    sudo dnf install git
    ```
2. Clone the repo:
    ```
    git clone https://github.com/eblanyrt/dotfiles-fedora-i3.git
    ```
3. Go to the directory, and launch the script:
    ```
    cd dotfiles-fedora-i3
    ./install.sh
    ```
4. Reboot the system.
